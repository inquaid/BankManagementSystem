## Files

Description of key files in the project:

- `Accounts.txt`: Stores account information including balances and transaction history.
- `CMakeLists.txt`: CMake configuration file for building the project.
- `CreateAccount.h`: Header file defining functions for creating new bank accounts.
- `Database.txt`: Database file for persistent storage of customer data.
- `InterFace.h`: Header file defining the user interface structure.
- `Login.h`: Header file containing login functionalities for bank employees and customers.
- `MyFunctions.exe`: Executable containing preliminary functions for account management.
- `MyFunctions.h`: Header file with additional utility functions.
- `README.md`: This documentation file.
- `SearchOnDatabase.h`: Header file for searching customer information in the database.
- `ShaTwoFiftySix.h`: Header file for implementing SHA-256 hashing for secure operations.
- `Username.txt`: File storing usernames for system access.
- `account.h`: Header file containing account-related functionalities.
- `header.h`: General header file for including common libraries and definitions.
- `main.c`: Main source file containing the main function and program entry point.
- `main.exe`: Main executable file after compilation.
- `sha256.h`: Header file implementing the SHA-256 hashing algorithm for data security.
